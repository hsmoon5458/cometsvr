INSTRUCTIONS FOR PARENT ANIMATION (RELATIVE POSITION ANIMATION)

1: KEEP ANIMATION CONTROLLER ON SAME BUTTON OBJECT (Cube in this package)

2: CREATE NEW EMPTY FOR BUTTON COORDINATES IN THE WORLD'S COORDINATE SYSTEM (CubeParent in this package)

3: MOVE BUTTON OBJECT UNDER (CubeParent) PARENT OBJECT

4: SET BUTTON (Cube) COORDINATES TO 0,0,0

5: MOVE **PARENT** (CubeParent) TO DESTINATION IN DESIRED COORDINATE SYSTEM

6: USE RELATIVE ANIMATION AS DESIRED
